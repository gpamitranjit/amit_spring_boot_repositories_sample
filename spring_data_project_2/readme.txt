steps to execute this application.

1. run this maven project in spring tool suite tool (altered version of Eclipse)
2. import "spring_data_project_2.json" file into your postman tool.
3. submit as many requests as you want whose request name is "Create todo records in todos table" from postman.
4. then submit the Get request "Get the record count whose name attribute has "sometodo" value"
5. the response of this last GET request will be count of all the records whose name attribute contains "sometodo" value
